### Solution
```json
[
  "Refrigerator",
  "Bookshelf",
  "Desk"
]
```
### Explanation
To solve this problem, we need to iterate through the list of items and filter out the ones that meet the specified criteria: category equals "appliances" and portable equals False. We can use a programming language like JavaScript to achieve this.

Here is the code:
```javascript
const items = [
  // ... (all the items listed in the problem statement)
];

const result = items.filter(item => item.category === "appliances" && !item.portable);

console.log(result);
```
This code uses the `filter()` method to create a new array that only includes the items that meet the specified criteria. The callback function `(item => item.category === "appliances" && !item.portable)` is applied to each item in the original array, and only the items that pass the test are included in the new array.

When we run this code, we get the following output:
```json
[
  "Refrigerator",
  "Bookshelf",
  "Desk"
]
```
This is the list of items that meet the specified criteria.
