### Step 2: Applying Transformation Rules

Now, let's apply the transformation rules to each record:

1. If `type` == "premium", then multiply `amount` by 1.1.
2. If `region` == "EU", then multiply `amount` by 1.2.
3. If `active` == False, then set `status` to 'archived'.

```python
import json

# Apply transformation rules
for record in records:
    if record["type"] == "premium":
        record["amount"] *= 1.1
    if record["region"] == "EU":
        record["amount"] *= 1.2
    if not record["active"]:
        record["status"] = "archived"

# Output the transformed records as a JSON array
output = json.dumps(records, indent=4)
print(output)
```

1. We iterate through each record in the `records` list.
2. For each record, we check the conditions of the transformation rules and apply the corresponding transformations.
3. After applying the transformations, we use `json.dumps()` to convert the list of transformed records into a JSON string.
4. The `indent=4` parameter is used to pretty-print the JSON with an indentation of 4 spaces.
