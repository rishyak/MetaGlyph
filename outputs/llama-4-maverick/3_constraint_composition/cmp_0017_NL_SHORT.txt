```python
filtered_dataset = [item for item in dataset if item['Priority'] == 'high' or item['Status'] != 'inactive']
```

### Step 3: Extract IDs and Count
Next, we extract the IDs from the filtered dataset and count the number of items.

```python
selected_ids = [item['ID'] for item in filtered_dataset]

```

